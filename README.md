# 项目概述

这是基于无名小站接口写的一个会员视频解析站点，基于LayUI布局。

# 项目运行

## a、克隆本项目 
克隆本项目并执行 `docker` 然后访问 `ip`, 需要域名自行解析

```docker
docker-compose up -d
```

## b、使用镜像

```
docker pull ghcr.io/xiaoxuan6/free_vip:latest
docker run --name free -p 80:80 -d ghcr.io/xiaoxuan6/free_vip:latest
```
